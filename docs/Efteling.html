Efteling

Implements the Efteling API framework.

new Efteling( [options])

Create new Efteling Object.
Parameters:
Name Type Argument Description
options Object <optional>
Properties
Name Type Argument Description
api_version String <optional>
 Version of the API to reference in request headers
digest_key String <optional>
 Key used to generate URL header digest
crypto_key String <optional>
 Key to decrypt wait times
crypto_cipher String <optional>
 Cipher to decrypt wait times
crypto_iv Buffer <optional>
 IV to decrypt wait times

 URL used for fetching POI data
